# TArenaUnity3D Legacy And Reference-Only Context Map

Status: active
Last updated: 2026-06-26

## Use When

Use this map only when the current user prompt, task, or brief explicitly asks
for legacy/reference material, migration comparison, old Retsot material, heavy
generated assets, binary files, backups, or historical design archaeology.

These sources are not normal routing context and must not be loaded by default.

## Explicit-Only Legacy References

Do not open these unless explicitly requested:

- `_codex/Context/Retsot GDD.md`
- `_codex/Context/Opisy jednostek invitational.pptx`
- `_codex/Gen_Im/RETSOT ONLINE/`
- zip archives, fonts, large generated images, and other heavy binary artifacts
  under `_codex/Gen_Im/`
- `_codex/git-backups/`

Retsot material may be used only as clearly labeled legacy reference. It is not
the current TArena project identity, not current design truth, and not default
agent context.

## Rule For New Legacy Routing

If another heavy or historical artifact needs to be documented, add it here as
explicit-only reference. Do not add it to the main context router or to default
agent startup instructions.
