Chat UI - proste ustawienia

Cel

Chcesz przesunac sam tekst w prawo wzgledem okna Chat. Nie przesuwaj calego Chat. Ustaw odstep na Content przez Vertical Layout Group > Padding > Left.

1. Chat

ElementUstawienieWartosc
Rect TransformAnchorsStretch / Stretch
Rect TransformScaleX 0.5, Y 0.5, Z 1
Scroll RectContentContent
Scroll RectViewportViewport
Scroll RectHorizontalOff, jesli chat ma przewijac tylko pionowo
Scroll RectVerticalOn
Scroll RectMovement TypeClamped

Pozycje Left, Right, Top, Bottom ruszaj tylko wtedy, gdy chcesz przesunac cale okno chatu.

2. Viewport

ElementUstawienieWartosc
Rect TransformAnchorsStretch / Stretch

ImageColorDowolny, moze byc przezroczysty
MaskShow Mask GraphicOff

Viewport powinien wypelniac Chat. Nie przesuwaj tekstu przez Viewport.

4. Text

ElementUstawienieWartosc
Rect TransformScaleX 1, Y 1, Z 1
TextBest FitOff
TextFont Size24-32 na start
TextHorizontal OverflowWrap
TextVertical OverflowOverflow
TextAlignmentLeft + Top albo Left + Middle
TextRaycast TargetOff, jesli tekst nie ma byc klikany

Nie przesuwaj Text recznie, jezeli jest dzieckiem Content z Vertical Layout Group. Layout i tak nadpisze jego pozycje.

Najkrotsza instrukcja

  1. Na Content ustaw Scale = 1, 1, 1.
  2. Na Content ustaw Vertical Layout Group > Padding > Left = 20.
  3. Na Text wylacz Best Fit.
  4. Na Text ustaw Font Size = 24.
  5. Nie ustawiaj pozycji Text recznie, bo robi to Vertical Layout Group.
