# Task Tracker Runbook

Use this for local markdown tasks, QA protocols, `/analyze`, `/implement`,
`/fix`, and `/close`.

## Workspace Boundary

This task tracker belongs to TArenaUnity3D.

Do not read, create, move, or close tasks from another project unless the user
explicitly asks for comparison or migration.

Chat/task titles must start with `[TARENA]`.

## Local Tracker

Active task files live directly in `_codex/tasks/`.

Completed or superseded tasks may live in `_codex/tasks/archive/`.

Analysis reports live in `_codex/tasks/Analysis/`.

Coding completion protocols and QA architecture reviews live in
`_codex/tasks/QA/`.

Do not create GitHub issues unless the user explicitly asks for that workflow.
