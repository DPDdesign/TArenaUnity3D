# Weekly Review

Read local production docs and active tasks. Separate done, partial, planned,
and scope-risk work. Recommend the next small playable step and one proof of
work if available.

## Project UI Text Rule

TArenaUnity3D uses TextMesh Pro only. If reviewed work includes UI or code text
references, treat TMP types such as `TMP_Text` and `TextMeshProUGUI` as the
required standard and legacy `UnityEngine.UI.Text` as drift.
